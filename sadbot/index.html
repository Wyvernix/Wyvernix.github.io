The !ip command

The ip command returns different ips depending on what server you are on. It will check the title of the broadcast for certain tags. The current ip will be updated every 5 minutes, and on demand.

!ip [servertag]

Returns the current ip (or matching ip if found)

!ip list

Lists triggers (server tags)

!ip add <trigger> <ip>

Trigger is the server name, eg. "mineplex". ip is the ip of the server, eg. "us.mineplex.com"
The trigger can be a RegEx. note: will be added as ".*?trigger.*?"

!!ip remove <trigger>

Removes the ip/trigger from the database
